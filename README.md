Queue-Modeling - WIP
==============

Is a command-line tool that automate the task of having to calculate
queue theory formulas

receives as input  λ and μ
and outputs the formula and answer

example
=======

./queue-modeling -lambda 10 -mu 15 

ρ = λ / μ = 0.67
